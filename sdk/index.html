SDK Access

To build an app on self-network its required to have our app installed so you can log in to the Self Developer Portal (production or sandbox).

SDK Host

When you start playing with Self you likely want to use Sandbox environment environment, as it’s easier to setup and less restrictive.

Please visit the Sandbox documentation page to download our mobile client from app stores.

Platform status and incidents

Project status page is available at status.joinself.com

Client libraries

Self offers official SDK libraries for different programming languages, which are regularly updated for breaking and non-breaking changes.

Self hosted HTTP service

In addition to the official SDKs we also provide a self hosted HTTP Self service that allows you to interact with the Self Network through a Rest API.
